Ticket handling application

My most complex semester work was created for a Java EE course. I decided to create a tickethandling application, that enables users to log in and manage user sent support requests. It connects to Postgre SQL database and runs on JBoss Application Server. The semester goal was to provide as much interfaces as possible, so I implemented a web-based UI with JSP, a REST API, JMS, JMX, local and remote EJBs and authentication services.
